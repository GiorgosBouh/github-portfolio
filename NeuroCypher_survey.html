Explore NeuroCypher ASD

May 30, 2026

NeuroCypher ASD is an AI-powered application that functions as both a screening assistant and an intelligent agent for detecting autism-related traits in toddlers. It combines graph-based knowledge representation, machine learning, and natural language processing to support research and educational efforts in early screening.

üåê Access the app: http://34.154.218.97:8501

Key Features:

  • Upload and analyze Q-Chat-10 and demographic screening data
  • Generate graph embeddings using Node2Vec
  • Predict ASD likelihood using a trained XGBoost model
  • Detect anomalies using Isolation Forest
  • Identify similar cases through graph-based connections
  • Ask natural language questions and receive Cypher-based answers
  • Visualize how traits and cases are interconnected in a knowledge graph

Why Use It?

  • Acts as an intelligent agent powered by AI and graphs
  • Implements a Graph-RAG (Retrieval-Augmented Generation) architecture
  • Provides explainable, context-aware outputs
  • Designed strictly for research and educational purposes
